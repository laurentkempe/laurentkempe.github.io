Windows Live Writer Beta 1

Aug 14, 2006

Microsoft just released a beta version of a new tool called Windows Live Writer Beta 1.

You might download it on this page: Microsoft Windows Live Writer

It took me a minute to install it and to configure it for my blog on weblogs.asp.net, you just need to know that the provider is Community Server and the url of the services is: http://weblogs.asp.net/blogs/metablog.ashx. Then you can go. I used it for that post. And it is really a great tool.

It seems that you can write plugins, but the link in the preferences dialog is going nowhere at the moment. But there is Windows Live Writer SDK. :-)

I will try it for some time bug my first impression is that it is REALLY GOOD!!!

You really have to try the web preview.

Update: I found two plugins from Tim Heuer on Codeplex, Tag4Writer and Flickr4Writer.
